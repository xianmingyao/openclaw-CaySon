## 重点项目介绍

### GitNexus
- **类型**：代码智能引擎
- **特点**：客户端知识图谱创建工具，完全在浏览器中运行
- **功能**：
  - 导入 GitHub 仓库或 ZIP 文件
  - 生成交互式知识图谱
  - 内置 Graph RAG Agent
- **适用场景**：代码探索
- **语言**：TypeScript
- **Stars**：7371

### LiteRT-LM（Google）
- **类型**：端侧大模型推理框架
- **特点**：大模型技术栈持续下沉
- **意义**：端侧推理性能的底层优化

### OpenCode
- **类型**：AI 编程工具
- **特点**：开源编程 Agent
