## 1. 🎯 这是什么（简介）

**扣子编程(code.coze.cn)** 是字节跳动推出的**Vibe Coding开发平台**，核心理念：

> "自然语言对话开发智能体、工作流、网页应用、移动应用，一键部署上线"

通过对话即可快速构建应用，告别部署烦恼。

---

## 2. 📝 关键功能点

### 核心能力

| 功能 | 说明 |
|------|------|
| **自然语言开发** | 对话即可构建智能体/工作流/网页/移动应用 |
| **一键部署** | 域名+云服务器+SSL证书+版本管理+监控 |
| **全栈集成** | 数据库、对象存储、AI模型、联网检索、文件处理 |
| **云端沙箱** | 专为AI Agent设计的开发环境，不怕本地环境损坏 |

### 支持类型

- 🖥️ 网页应用
- 📱 移动应用
- 📲 小程序
- 🤖 智能体
- ⚙️ 工作流
- 🔧 技能

## 3. ⚡ 怎么使用

### 访问地址
- 扣子编程：https://code.coze.cn/home
- 扣子智能体：https://www.coze.cn

### 工作方式
```python
# 输入自然语言描述需求
"帮我做一个Todo List应用，支持任务优先级和截止时间提醒"

# AI自动生成完整应用
# 一键部署上线
```

### 示例项目
- Todo List应用（支持优先级和截止时间）
- 番茄钟网站（专注统计+每日目标）
- 个人博客（文章分类+全文搜索）
- 记账工具（收支分类+月度图表）
- AI聊天页面（会话保存+一键清空）

### 抖音评论区炸裂
> **作者回复**：
> "一个是龙虾的逻辑，一个是Claude cowork逻辑。扣子处于中间，像龙虾一样灵活，又不像龙虾一样难用"

### OpenClaw vs 扣子对比

| 维度 | OpenClaw(龙虾) | 扣子(Coze) |
|------|----------------|-------------|
| **定位** | 多渠道AI助手平台 | Vibe Coding开发平台 |
| **用户友好性** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **灵活性/可控性** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| **部署方式** | 自托管 | 一键云端部署 |
| **价格** | 免费（自托管） | 可能收费 |

### 结论
- **扣子适合**：快速原型、零基础用户、企业内部工具
- **OpenClaw适合**：高度定制、需要多渠道、安全敏感
