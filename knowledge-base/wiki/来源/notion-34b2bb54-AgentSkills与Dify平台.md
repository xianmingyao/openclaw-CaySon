# AgentSkills与Dify平台

> 类型：概念/方法论

> 更新时间：2026-04-10

## 简介

基于Dify平台的AgentSkills使用，从工作流到Agent的AI应用模式进化。

## AI应用模式演进

| 阶段 | 核心 | 局限/优势 |

|------|------|-----------|

| Chat | 自然语言交互 | 仅文本，无法交付成果 |

| Workflow | 大模型+工具代码 | 逻辑写死，不够"聪明" |

| Agent | 智能体 | 自主决策，动态调用 |

## SKI架构

AgentSkills + Dify工作流 = Skill + Knowledge + Instruction

## 三大核心优势

1. **无限扩展** - 可扩展的业务能力

2. **易于维护** - 模块化设计

3. **私有化友好** - 企业私有部署

## 总结

- 从"拼工具" → "拼智能体能力"

- 自然语言(Markdown)定义复杂业务逻辑

- 一套架构适配无限业务场景
