## 📝 Mem0 核心功能

### 1. 语义搜索

# 搜索用户偏好

node scripts/mem0-search.js "user preferences" --limit=3

# 搜索特定用户

node scripts/mem0-search.js "query" --user=username --limit=5

**技术原理：**

- 向量嵌入（Embedding）

- 余弦相似度

- Milvus/ChromaDB 存储

### 2. 记忆存储

# 显式存储

node scripts/mem0-add.js "用户喜欢简洁的更新"

# 对话存储（自动提取）

node scripts/mem0-add.js --messages='[{"role":"user","content":"..."}]'

**智能提取：**

- LLM 自动去重

- 合并相似记忆

- 提取关键偏好

## 📝 Self-Improving 核心功能

### 1. 三层存储架构

HOT (memory.md)     ≤100行  常驻内存

↓

WARM (projects/domains/)  ≤200行  按需加载

↓

COLD (archive/)     无限    归档存储

### 2. 学习信号

**✅ 应该记录：**

- 用户纠正："不，应该是..."

- 偏好信号："我喜欢..."

- 模式候选：重复3次的操作

**❌ 不要记录：**

- 一次性指令

- 特定上下文

- 假设性内容

### 3. 自我反思流程

完成工作 → 暂停评估 → 发现问题 → 记录教训

↓

教训重复3次 → 提升到 HOT 层

**反思格式：**

CONTEXT: [任务类型]

REFLECTION: [发现的问题]

LESSON: [下次改进]

### 4. 自动晋升/降级

| 条件 | 动作 |

|------|------|

| 7天内使用3次 | 提升到 HOT |

| 30天未使用 | 降级到 WARM |

| 90天未使用 | 归档到 COLD |
