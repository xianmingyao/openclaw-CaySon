# Agent

> 分类：概念
> 创建时间：2026-04-09 16:09

在计算机科学中，“Agent”通常指能够自主执行任务或决策的软件程序或机器人。它具备感知环境、制定策略和采取行动的能力。关键要点包括：自治性（能独立工作）、反应性（对环境变化迅速响应）、社会性（与其他智能体交互）以及适应性（根据经验学习并改进）。在人工智能领域，Agent常用于模拟真实世界中的决策过程，广泛应用于游戏开发、机器人技术及自动化系统中。
