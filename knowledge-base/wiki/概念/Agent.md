# Agent

> 分类：概念
> 创建时间：2026-04-09 15:37

在计算机科学领域中，“Agent”是指一种能够感知环境、自主做出决策并执行相应行动的人工智能实体。它具备感知与作用于环境的能力，并能根据自身的目标或规则进行行为选择。关键要点包括：自治性（能够独立运作）、适应性（对环境变化作出响应）、社会性（与其他agent交互合作）以及反应性（即时回应环境刺激）。在不同的应用场景中，Agent可以是虚拟助手、机器人或是复杂的智能系统。
