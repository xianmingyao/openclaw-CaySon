# Agent

> 分类：概念
> 创建时间：2026-04-11 20:02

在计算机科学领域中，“Agent”通常指一种能够自主执行任务或决策的人工智能实体。它具有感知环境、制定策略并采取行动的能力。关键要点包括：自治性（能独立运作）、反应性（对环境变化迅速响应）、社会性（与其他Agent交互合作）和适应性（根据经验学习改进）。在不同应用场景中，如机器人技术、网络系统管理或虚拟助手开发等，“Agent”扮演着重要角色。
