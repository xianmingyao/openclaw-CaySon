# Agent

> 分类：概念
> 创建时间：2026-04-09 20:03

在计算机科学领域中，“Agent”通常指一种能够感知环境并根据特定目标采取行动的人工智能系统。它具有自主性、适应性和反应性等特性，并且能够在复杂环境中执行任务或解决问题。关键要点包括：自主性（能独立做出决策）、适应性（能够调整策略以应对变化的环境）和反应性（对环境刺激作出快速响应）。此外，Agent还可以分为理性Agent与非理性Agent，前者追求最大化效用函数，后者则不遵循这一原则。
