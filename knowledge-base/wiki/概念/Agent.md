# Agent

> 分类：概念
> 创建时间：2026-04-12 20:03

在计算机科学中，“Agent”通常指一种能够自主执行任务或决策的人工智能实体。它具有感知环境、制定策略并采取行动的能力。关键要点包括：自治性（能独立工作）、反应性（对环境变化迅速响应）、社会性（与其他agent交互）和适应性（根据经验学习和改进）。Agent技术广泛应用于机器人、网络安全、游戏AI等领域，是实现智能系统的重要组成部分。
