# Agent

> 分类：概念
> 创建时间：2026-04-13 20:00

在计算机科学领域中，“Agent”通常指能够自主执行任务或决策的人工智能实体。它具备感知环境、制定策略以及采取行动的能力，并能根据目标调整行为。关键要点包括：自治性（能够独立运作）、反应性（对环境变化作出即时响应）、社会性（与其他代理交互）和适应性（学习并改进自身性能）。Agent技术广泛应用于机器人、虚拟助手及智能系统等领域，是现代人工智能研究的重要组成部分。
