# Agent

> 分类：概念
> 创建时间：2026-04-09 15:28

在计算机科学中，“Agent”通常指能够自主执行任务或决策的软件程序或机器人。它具备感知环境、制定策略并采取行动的能力。关键要点包括：自治性（能独立工作）、反应性（对变化快速响应）、社会性（与其他代理交互）和适应性（根据经验学习）。在人工智能领域，Agent常用于模拟真实世界中的智能行为，支持从简单规则引擎到复杂的机器学习模型的应用。
