# Control Loop

> 分类：概念
> 创建时间：2026-04-09 15:29

Control Loop（控制回路）是一种自动化系统，用于维持一个或多个物理量在预设的目标值附近。它由测量装置、控制器和执行器三部分组成，通过反馈机制调整输入以抵消干扰并达到目标状态。关键要点包括：闭环结构确保了系统的稳定性与准确性；PID算法是常用的控制策略之一；响应时间和误差范围是评价控制回路性能的重要指标。
