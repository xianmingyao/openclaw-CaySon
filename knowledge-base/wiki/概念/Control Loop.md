# Control Loop

> 分类：概念
> 创建时间：2026-04-09 20:03

Control Loop（控制回路）是一种自动控制系统，通过闭环反馈机制调整系统输出以达到预期目标。它由测量元件、控制器、执行器和被控对象组成。关键要点包括：设定点与实际值的比较产生误差信号；控制器根据此信号调整控制量；执行器作用于被控对象改变其状态；最终使系统输出接近设定值，形成闭环调节过程。控制回路广泛应用于工业自动化、航空航天及医疗设备等领域。
