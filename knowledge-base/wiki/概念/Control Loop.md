# Control Loop

> 分类：概念
> 创建时间：2026-04-13 20:00

Control Loop（控制回路）是一种自动控制系统，通过反馈机制监测系统输出并调整输入以维持设定的目标值或状态。其关键组成部分包括控制器、传感器、执行器以及被控对象。控制回路由测量环节、比较环节、调节环节和放大环节组成，能够实现精确的动态响应与稳定性能。在工业自动化、航空航天及医疗设备等领域广泛应用。
