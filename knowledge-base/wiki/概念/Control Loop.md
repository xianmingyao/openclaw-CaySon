# Control Loop

> 分类：概念
> 创建时间：2026-04-12 20:03

Control Loop（控制回路）是一种自动化系统中用于维持特定参数稳定性的闭环反馈机制。它由测量装置、控制器和执行器三部分组成，通过比较实际值与设定值之间的差异来调整输出，以达到预期的工艺状态或性能目标。关键要点包括：1) 反馈是控制回路的核心；2) 常见类型有PID（比例-积分-微分）控制；3) 应用于工业过程控制、机器人技术及家电设备等众多领域。
