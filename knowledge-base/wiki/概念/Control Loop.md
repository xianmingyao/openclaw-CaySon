# Control Loop

> 分类：概念
> 创建时间：2026-04-10 20:03

Control Loop（控制回路）是一种自动化系统的关键组成部分，用于维持特定参数在设定值附近稳定运行。它通常由传感器、控制器和执行器三部分组成：传感器监测被控对象的状态并反馈给控制器；控制器根据预设逻辑处理信息并与期望值比较后发出指令至执行器进行调整；执行器则负责改变系统的物理状态以达到控制目标。关键要点包括闭环反馈机制确保系统动态响应的快速性和准确性，以及PID（比例-积分-微分）控制算法在工业应用中的广泛使用。
