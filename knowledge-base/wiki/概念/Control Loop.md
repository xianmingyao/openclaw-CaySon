# Control Loop

> 分类：概念
> 创建时间：2026-04-09 16:09

Control Loop（控制回路）是一种自动化系统的核心组成部分，用于维持一个或多个过程变量在预设的目标值附近。它通常包括测量、比较、计算和执行四个基本环节：首先通过传感器测量实际值；然后与设定值进行对比以确定偏差；接着控制器根据偏差大小调整控制量；最后执行器依据控制信号对被控对象施加影响，使系统输出接近目标值。关键要点在于闭环反馈机制确保了系统的稳定性和准确性，广泛应用于工业自动化、航空航天和医疗设备等领域。
