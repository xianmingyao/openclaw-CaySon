# Feedback System

> 分类：概念
> 创建时间：2026-04-13 20:00

Feedback System：一种控制系统，通过将系统输出的一部分或全部返回到输入端来影响其行为，以达到预期目标。关键要点包括：闭环控制、误差修正、稳定性与响应速度。反馈机制可以是正反馈（增强偏差）或负反馈（减少偏差），广泛应用于工程、生物、经济等多个领域。
