# Feedback System

> 分类：概念
> 创建时间：2026-04-09 15:37

Feedback System：一种控制系统，通过将系统输出的一部分或全部返回到输入端与参考输入进行比较，从而调整系统的性能以达到预期目标。关键要点包括：闭环控制、误差校正、稳定性与动态响应、反馈类型（正面/负面）、调节器设计及应用广泛于工业自动化、航空航天和生物医学工程等领域。
