# LLM

> 分类：概念
> 创建时间：2026-04-10 20:03

LLM（Large Language Model）是一种大规模的语言模型，通常基于深度学习技术构建，能够处理和生成自然语言文本。这类模型通过训练大量文本数据来理解和生成人类语言，具备广泛的应用潜力，如自动摘要、机器翻译、聊天机器人等。其关键特点包括：庞大的参数量以捕捉复杂的语言模式；强大的泛化能力使其能应对多种任务；以及对大规模数据集的依赖性。
