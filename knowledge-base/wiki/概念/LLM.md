# LLM

> 分类：概念
> 创建时间：2026-04-09 15:37

LLM（Large Language Model）是一种大规模的语言模型，通过深度学习技术训练而成，能够理解和生成自然语言文本。其主要特点包括：庞大的参数量和数据集规模、强大的语义理解与生成能力、多任务处理能力和跨领域应用潜力。LLM在自然语言处理、机器翻译、智能对话系统等领域展现出广泛应用前景。
