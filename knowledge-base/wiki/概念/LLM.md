# LLM

> 分类：概念
> 创建时间：2026-04-12 20:03

LLM（Large Language Model）是一种大规模的语言模型，通常指训练规模巨大、参数量超过十亿甚至更多的人工智能模型。这类模型通过深度学习技术，在大量文本数据上进行训练，能够生成连贯的自然语言文本，并具备一定的理解和生成能力。关键要点包括：强大的语言理解与生成能力；需要大量的计算资源和高质量的数据支持；在多个自然语言处理任务中表现出色，如机器翻译、问答系统等。
