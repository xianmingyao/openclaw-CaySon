# LLM

> 分类：概念
> 创建时间：2026-04-09 16:10

LLM（大型语言模型）是一种能够处理和生成自然语言文本的人工智能系统。这类模型通常基于深度学习技术，拥有庞大的参数量和训练数据集，使其在多种自然语言任务上表现出色。其关键特点包括强大的语义理解和生成能力、多样的应用场景以及对上下文的理解与记忆能力。LLM的发展推动了诸如机器翻译、文本摘要、对话系统等领域的进步，并且正在逐步改变人机交互的方式。
