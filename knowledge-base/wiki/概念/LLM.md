# LLM

> 分类：概念
> 创建时间：2026-04-13 20:00

LLM（Large Language Model）是一种大规模的语言模型，通过深度学习技术训练而成，能够理解、生成人类自然语言文本。其关键特点包括：庞大的参数量以捕捉复杂的语言结构；广泛的数据集训练以覆盖多种语言现象；强大的上下文理解和生成能力。这些特性使得LLM在机器翻译、自动摘要、对话系统等领域展现出巨大潜力和应用价值。
