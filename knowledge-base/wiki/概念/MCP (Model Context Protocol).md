# MCP (Model Context Protocol)

> 分类：概念
> 创建时间：2026-04-09 16:10

MCP（Model Context Protocol）是一种用于软件开发中模型与视图之间交互的协议。它定义了如何在不同的上下文环境中管理数据模型，确保数据的一致性和完整性。关键要点包括：1) 强调数据模型的中心地位；2) 支持多视角的数据展示和操作；3) 促进模块化设计，提高代码复用性与可维护性；4) 简化复杂系统的开发过程，增强团队协作效率。
