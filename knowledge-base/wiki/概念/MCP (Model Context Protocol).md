# MCP (Model Context Protocol)

> 分类：概念
> 创建时间：2026-04-09 20:03

MCP（Model Context Protocol）是一种用于软件开发中的数据模型与业务逻辑之间交互的协议。它允许在不同的模块或组件间高效地传递和处理复杂的数据结构，确保数据的一致性和完整性。关键要点包括：1. 提高系统的可维护性和扩展性；2. 通过定义明确的数据接口减少耦合度；3. 支持跨平台和多语言开发环境的集成与协作。
