# MCP (Model Context Protocol)

> 分类：概念
> 创建时间：2026-04-11 20:03

MCP（Model Context Protocol）是一种设计模式，用于在软件开发中明确模型与视图之间的交互方式。它强调通过协议定义数据模型如何响应用户的操作或事件，确保用户界面能够根据模型的状态变化实时更新。关键要点包括：定义清晰的接口以减少视图对模型内部实现的依赖；促进代码的解耦和重用；提高系统的灵活性和可维护性。MCP 常用于需要复杂状态管理的应用中，如桌面应用或大型Web应用程序。
