# MCP (Model Context Protocol)

> 分类：概念
> 创建时间：2026-04-10 20:03

MCP（Model Context Protocol）是一种用于软件开发中的协议，旨在规范应用程序中模型与上下文之间的交互方式。通过明确模型如何响应不同的操作请求及其对环境变化的反应机制，MCP 提升了系统的可维护性和扩展性。其核心在于定义清晰的接口和行为规则，确保不同模块间的高效协作。关键要点包括：1）分离关注点，增强组件独立性；2）简化状态管理与数据流动；3）促进代码重用与模块化设计。
