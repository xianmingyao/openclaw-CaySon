# MCP (Model Context Protocol)

> 分类：概念
> 创建时间：2026-04-12 20:03

MCP（Model Context Protocol）是一种设计模式，用于在软件开发中明确模型与视图之间的交互方式。它强调通过定义清晰的接口来传递数据上下文，确保视图能够准确地反映模型的状态变化，同时保持两者间的解耦。关键要点包括：1) 促进模型和视图之间的松散耦合；2) 提高代码的可维护性和扩展性；3) 确保数据状态的一致性更新机制。
