# Prompt Engineering

> 分类：概念
> 创建时间：2026-04-12 20:03

Prompt Engineering 是一种技术，旨在通过设计有效的提示来优化人机交互的质量与效率。它涉及理解用户需求、明确任务目标以及构建简洁明了的指令，以促使AI系统生成更准确、更有针对性的响应。关键要点包括：1）明确性：确保提示清晰无歧义；2）简洁性：避免冗余信息，保持简练；3）灵活性：根据不同的应用场景调整提示内容；4）反馈循环：利用结果优化后续提示设计。
