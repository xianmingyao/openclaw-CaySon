# Prompt Engineering

> 分类：概念
> 创建时间：2026-04-11 20:03

Prompt Engineering 是一种通过设计、优化与迭代提示（prompt）来引导AI系统产生更准确、更有针对性输出的技术。关键要点包括：明确性（确保指令清晰）、简洁性（保持信息精炼）、多样性（探索不同表达方式以获得最佳结果）。此技术广泛应用于自然语言处理领域，旨在提高人机交互的质量和效率。
