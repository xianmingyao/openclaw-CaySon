# Prompt Engineering

> 分类：概念
> 创建时间：2026-04-09 20:03

Prompt Engineering 是一种技术，旨在通过精心设计的提示（prompt）来优化人机交互，特别是在自然语言处理领域。它涉及创建能够引导模型产生准确、相关且有用响应的指令或问题。关键要点包括：明确性与简洁性、意图导向、多样性与创造性以及持续迭代以提升性能。Prompt Engineering 的目标是提高生成式AI系统的效率和效果，使其更好地满足用户需求。
