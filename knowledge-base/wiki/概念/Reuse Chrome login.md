# Reuse Chrome login

> 分类：概念
> 创建时间：2026-04-10 08:13

Reuse Chrome Login 是一种功能，允许用户在多台设备上使用同一组 Google 账户信息登录 Chrome 浏览器。通过启用此功能，用户可以在不同设备间无缝切换，保持浏览历史、书签及其他设置的一致性。关键要点包括：增强跨设备同步体验；提高隐私保护；简化多设备间的账号管理流程。
