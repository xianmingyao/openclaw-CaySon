## 🎯 这是什么（简介）

**Mythos** 是 Anthropic Claude 系列的下一代神秘模型，代号可能与"神话"相关。据评论区透露，**zueshammer** 是一个融合了 OpenClaw + Hermes + Claude Code 的超级智能体项目。
