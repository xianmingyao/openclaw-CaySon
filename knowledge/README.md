## 🔍 如何使用

- 直接阅读 `.md` 文件获取知识
- 内容按主题分类，支持全文搜索
- 新学到的内容及时追加到对应文件

## 📝 贡献知识

遇到有价值的内容时：
1. 判断类别（项目/技术/工具/经验）
2. 找到或创建对应的 `.md` 文件
3. 按格式追加内容
4. 更新本索引文件

## ⚠️ 注意事项

- 知识库放在 workspace 内，随 git 同步
- 敏感信息不要写入（如密钥、密码）
- 定期整理，避免信息过载
