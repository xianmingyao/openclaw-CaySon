## 核心功能

| 功能 | 说明 |
|------|------|
| **主题转课堂** | 把任意主题/文档变成沉浸式AI课堂 |
| **交互幻灯片** | 自动生成PPT风格的课件 |
| **测验题目** | 自动出题测试理解 |
| **多角色讨论** | 学生/老师/助教多个AI角色互动 |
| **课堂视频** | 一键生成完整长课堂视频 |
| **实时问答** | AI回答学生问题 |
| **📄 PDF转课堂** | 一键将PDF转化为包含AI教师+白板+测验的沉浸式学习场景（Vivi 2.0新功能）|
| **🎓 AI教师** | 虚拟AI教师角色，模拟真实课堂互动 |
| **📊 白板互动** | 课堂画图解释，帮助理解复杂概念 |

## 安装部署

### 方式1: Docker一键部署（推荐）

```bash
# 克隆仓库
git clone https://github.com/THU-MAIC/OpenMAIC.git
cd OpenMAIC

# Docker启动
docker-compose up -d

# 访问
# http://localhost:3000
```

### 方式2: 本地安装

```bash
# Python环境要求
python >= 3.10

# 安装依赖
pip install openmaic

# 启动
python -m openmaic.app
```

## 使用流程

### 1. 创建课堂

```bash
# 方式1: Web界面
# 访问 http://localhost:3000
# 输入主题："机器学习基础"
# 选择：课程长度、难度、角色数量

# 方式2: 命令行
openmaic create --topic "机器学习基础" --duration long
```

### 2. 生成内容

```bash
# 输入源类型
# - 主题描述
# - PDF文档
# - 视频文件
# - 网页链接

# 示例：从PDF生成
openmaic generate --input lecture.pdf --type pdf
```

### 3. 生成课堂视频

```bash
# 生成完整课堂视频
openmaic video --duration 45 --style lecture

# 参数
# --duration: 视频时长（分钟）
# --style: lecture/classroom/debate
# --agents: 参与角色数量
```

## 演示案例

### 案例：生成"人工智能导论"课堂

```bash
# Step 1: 创建项目
openmaic create --topic "人工智能导论" --level beginner

# Step 2: 选择组件
# ✓ 交互幻灯片
# ✓ 随堂测验
# ✓ 问答环节
# ✓ 小组讨论
# ✓ 生成视频

# Step 3: 生成
openmaic generate --duration 45 --video

# 输出：
# - slides.pdf (课件)
# - quiz.json (测验题)
# - classroom.mp4 (45分钟课堂视频)
```

### 案例：上传课件生成课堂

```bash
# 上传已有PPT/PDF
openmaic upload --file lesson.pdf

# AI自动分析内容
# 生成多角色互动
# 输出一整套课堂材料
```

## 优缺点

| ✅ 优点 | ❌ 缺点 |
|---------|---------|
| 一键生成完整课堂 | 生成内容需要AI API |
| 多角色真实互动 | 视频质量依赖AI模型 |
| 内置测验系统 | 需要较强硬件配置 |
| 清华背书，品质保证 | 2年以上迭代仍在进行 |
| 开源免费 | 中文资料较少 |

| **学术验证(JCST 2026)** | 700+学生验证，品质可靠 |
