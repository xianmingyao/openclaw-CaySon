# Skills 知识库

## 目录结构

```
skills/
├── shared/        # 通用技能（跨项目可用）
├── ningsk/       # 宁兄专属技能（基于宁兄工作流）
└── README.md     # 本文件
```

## 技能沉淀规则

### 触发条件
满足以下任一条件时，生成技能文档：
1. 复杂任务成功完成（>30分钟）
2. 重复任务超过3次
3. 踩坑记录（避坑指南）
4. 宁兄明确要求沉淀

### 命名规范
```
yyyy-MM-dd-技能名称.md
例：2026-04-15-抖音视频扒取.md
```

### 技能文档模板

```markdown
