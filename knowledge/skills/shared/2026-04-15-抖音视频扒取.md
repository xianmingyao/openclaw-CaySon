# 抖音视频扒取

## 一句话描述
通过 agent-browser 自动化扒取抖音视频信息（标题、描述、评论、链接等）

## 触发场景
- 用户发来抖音链接需要提取信息
- 扒取视频章节内容
- 获取评论区GitHub项目地址

## 使用方法

### 前置条件
- `npx agent-browser` 可用
- Chrome已开且登录状态（如需查看登录后内容）

### 核心步骤
1. **打开视频页面**
   ```bash
   npx agent-browser open "https://v.douyin.com/xxx/"
   # 或直接用视频ID
   npx agent-browser open "https://www.douyin.com/video/{视频ID}"
   ```

2. **等待页面加载**（等待3-5秒）

3. **提取页面内容**
   ```bash
   # 提取文本内容
   npx agent-browser eval "JSON.stringify({text: document.body.innerText.substring(0,5000)})"
   
   # 提取链接（筛选GitHub）
   npx agent-browser eval "JSON.stringify({links: Array.from(document.querySelectorAll('a[href*=\"github\"]')).map(a => ({text: a.innerText, href: a.href}))})"
   ```

4. **截图保存**
   ```bash
   npx agent-browser screenshot
   ```

### 命令清单
```bash
# 打开视频
npx agent-browser open "URL" --headed

# 等待
npx agent-browser wait 3000

# 获取当前URL
npx agent-browser get url

# 滚动页面
npx agent-browser scroll 2000

# 提取全文本
npx agent-browser eval "JSON.stringify({text: document.body.innerText.substring(0,6000)})"

# 提取特定链接
npx agent-browser eval "JSON.stringify({links: Array.from(document.querySelectorAll('a[href*=\"github\"]')).map(a => ({text: a.innerText.substring(0,80), href: a.href}))})"

# 截图
npx agent-browser screenshot
```

## 注意事项

### ⚠️ 常见问题

1. **SIGKILL问题**
   - 抖音页面加载时容易被系统SIGKILL
   - 解决：直接用视频ID打开，避免重定向

2. **评论区内嵌搜索**
   - "karpathy知识库开源地址"这类是站内搜索链接，不是真实GitHub
   - 需要结合评论区内容手动确认

3. **动态加载内容**
   - 页面可能需要滚动触发懒加载
   - 使用 `npx agent-browser scroll 2000` 加载更多

4. **agent-browser daemon问题**
   - daemon已在跑时`--headed`参数会被忽略
   - 解决：先 `npx agent-browser close` 再重新打开

## 思考过程

- **遇到的问题**：抖音页面直接访问会被SIGKILL，需要通过短链解析后的真实视频ID访问
- **最初的想法**：直接用curl/wget抓取，但抖音反爬很强
- **为什么成功**：使用真实Chrome渲染，天然绕过反爬
- **下次遇到类似问题**：优先用agent-browser，比curl稳10倍
