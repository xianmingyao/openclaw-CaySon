# 架构图生成 fireworks-tech-graph

## 一句话描述
用自然语言描述系统架构，自动生成SVG+PNG技术图

## 触发场景
- 用户需要画架构图、流程图
- 生成知识库文档配图
- 项目汇报需要可视化
- 关键词：画图、架构图、流程图、可视化

## 前置条件
```bash
# 安装依赖 (macOS)
brew install librsvg

# 验证
rsvg-convert --version
```

## 安装方式
```bash
# 克隆到本地
git clone https://github.com/yizhiyanhua-ai/fireworks-tech-graph.git ~/.claude/skills/fireworks-tech-graph

# 或用npx skills
npx skills add yizhiyanhua-ai/fireworks-tech-graph
```

## 核心用法

### 基本触发
```
画一张 RAG 流程图
```

### 指定风格
```
画一张微服务架构图，风格2（暗黑极客风）
```

### 指定输出
```
生成 Mem0 架构图，输出到 ~/Desktop/
```

## 内置AI/Agent Pattern

```
RAG Pipeline         → Query → Embed → VectorSearch → Retrieve → LLM → Response
Agentic RAG          → RAG + Agent循环 + 工具调用
Agentic Search       → Query → Planner → [Search/Calc/Code] → Synthesizer
Mem0 记忆层         → Input → Memory Manager → [VectorDB + GraphDB] → Context
Agent 记忆类型       → 感知 → 工作 → 情景 → 语义 → 程序记忆
Multi-Agent          → Orchestrator → [SubAgent×N] → Aggregator
Tool Call 流程       → LLM → Tool Selector → Execution → Parser → LLM
```

## 思考过程

- **遇到的问题**：手动画架构图费时费力
- **最初想法**：用Mermaid.js，但语法复杂
- **为什么成功**：自然语言输入，AI理解后生成SVG，无需手写代码
- **下次遇到类似问题**：优先用这个工具，比手画快100倍
