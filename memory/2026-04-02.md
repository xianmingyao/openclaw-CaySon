### 1. 抖音大力AI视频研究 - 前端自动化工具集

**来源：** 抖音@大力AI（2026-03-22发布，5万+点赞）

### 2. browser-use CLI 2.0 深度测试

**安装：** `pip install browser-use` + `playwright install chromium`

**实测结果：**
| 功能 | 状态 |
|------|------|
| `browser-use --version` | ✅ 0.12.5 |
| `browser-use open "url"` | ✅ 成功 |
| `browser-use state` | ✅ DOM树输出 |
| `browser-use eval` | ✅ JS执行成功 |
| `browser-use screenshot` | ✅ 成功 |
| `browser-use python` | ✅ Python执行 |
| `browser-use close` | ✅ 成功 |
| `browser-use extract` | ❌ 未实现 |
| `browser-use --mcp` | ⚠️ 待进一步测试 |

**避坑：**
- Windows必须设置 `$env:PYTHONIOENCODING="utf-8"`
- browser-use install 失败 → 用 `playwright install chromium`

### 3. WebMCP 研究

**来源：** Chrome官方 2026-02-10 发布

**核心概念：**
- 网站暴露结构化工具给AI Agent
- 声明式（HTML）+ 命令式（JS）两种API
- 比截图方式节省 ~89% tokens
- Chrome 146+ Canary 支持，正式版2026年中

### 4. mcp-browser-use 测试

**状态：** API不兼容，暂不可用
- `pip install mcp-browser-use` → 0.1.5
- 报错：`ModuleNotFoundError: No module named 'browser_use.browser.browser'`
- 原因：browser-use 0.12.5 API结构变化
