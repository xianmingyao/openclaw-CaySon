### Bug 5 修复：aiohttp 资源泄露

**问题**：`asyncio.run(master.publish())` 后未调用 `close()`，导致 `Unclosed client session` 警告

**修复方案**：在 `cli.py` 的 publish 命令中添加 `try/finally` 确保资源关闭

```python
# cli.py - publish 命令
try:
    result = asyncio.run(master.publish(product_name=product_info, context="jingmai"))
finally:
    # 确保资源清理
    if hasattr(master, 'ollama_service'):
        await master.ollama_service.close()
```
