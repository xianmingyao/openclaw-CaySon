## 快速开始
```bash
# 安装依赖
pip install openai pillow

# 生成图片
python generate.py "一只可爱的猫咪在咖啡店里喝咖啡，动漫风格"
```

## 配置
```json
{
  "api_key": "your-api-key",
  "model": "dall-e-3",
  "default_size": "1024x1024",
  "default_quality": "standard"
}
```

## 使用示例
1. **文生图**：输入描述 → 生成图片
2. **图生图**：上传参考图 → 生成变体
3. **风格迁移**：选择风格 → 应用到图片
4. **批量生成**：输入多个描述 → 批量生成
