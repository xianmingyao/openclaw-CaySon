# 剪映

用途与边界

- 面向公开官网、模板页、功能页、帮助中心与定价页的信息整理
- 不提供账号代操作、素材版权绕过、付费资源下载或接口调用能力
- 仅处理公开可见的模板、编辑能力、导出规格、帮助信息与套餐说明

关键入口

- 主页：https://www.capcut.cn/
- 功能与模板：官网中的模板中心、创作能力、素材或编辑器说明入口
- 帮助与定价：帮助中心、版权说明、套餐页与更新说明入口

常见任务

- 模板类型、编辑能力、导出规格与协作能力整理
- 素材、版权、商用说明与帮助信息汇总
- 套餐、会员权益、限制条件与更新日志提取

数据字段

- 模板类型、编辑功能、导出格式、分辨率或规格、链接
- 版权说明、商用边界、帮助页、素材入口链接
- 套餐名称、价格、限制项、协作功能差异

自动化要点

- 优先使用官方模板页与帮助页，不依赖第三方教程作为事实来源
- 版权和商用说明必须保留原始链接，避免误用素材
- 不处理用户私有项目、未公开素材库或账号资产

示例流程

- 功能整理：访问 Jianying 功能页 → 提取创作能力与导出规格 → 输出摘要
- 模板汇总：进入模板中心 → 抽取分类与代表入口 → 输出链接清单
- 权益整理：访问定价与版权页 → 提取套餐与商用边界 → 输出表格

合规提示

- 遵守平台规则、版权要求与账号安全要求
- 只整理公开信息，不输出敏感数据或非公开内容
