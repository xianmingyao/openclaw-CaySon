### 4. 云部署步骤

#### 阶段1：API 集成（1-2天）

```bash
# 文生图 API
pip install replicate
replicate run stability-ai/sdxl:...  # Stable Diffusion XL

# 图生视频 API  
pip install runway-python
# 或
curl -X POST https://api.runwayml.com/v1/video generation
```

#### 阶段2：编排服务（3-5天）

```python
# pipeline.py
async def generate_video(prompt: str, style: str):
    # 1. 文生图
    image = await text_to_image(prompt, style)
    
    # 2. 图生视频
    video = await image_to_video(image)
    
    # 3. 剪辑合成
    final = await compose_video(video, prompt)
    
    return final

# 使用 FastAPI 包装
app = FastAPI()
app.post("/generate")(generate_video)
```

#### 阶段3：部署上线（2-3天）

```bash
# Modal 部署
modal deploy pipeline.py

# 或 Docker + Sagemaker
docker build -t video-pipeline .
docker push ecr.aws/...
```
